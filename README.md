# Web Server Hardening

This project demonstrates installation and basic security hardening of a web server.

## Features
- Apache installation
- HTTPS configuration
- Firewall setup
- Fail2ban protection
- Security config tweaks

## Tools used
- Ubuntu Server
- Apache
- UFW (firewall)
- Fail2ban

## Learning goals
- Install a web server from scratch
- Harden server against attacks
- Automate installation and security with scripts
