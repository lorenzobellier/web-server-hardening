# Lab Setup

## Environment
- Ubuntu Server VM
- Local network
- Root access required

## Steps
1. Run install_web.sh to install Apache and firewall
2. Run secure_web.sh to configure fail2ban and secure Apache
3. Test access via browser (HTTP/HTTPS)
4. Verify firewall and fail2ban are running

## Learning Goals
- Understand web server installation
- Apply basic hardening techniques
- Document procedures for professional use
